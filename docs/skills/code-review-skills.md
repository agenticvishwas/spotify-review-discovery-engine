# Code Review Skills

## Purpose
Defines best practices for Code Review Skills in the Spotify Review Discovery Engine.

## Objectives
- Align with fellowship problem statement
- Produce maintainable, evidence-backed implementations
- Support modular AI workflows

## Principles
- Prefer structured outputs over free text
- Preserve traceability to source reviews
- Document assumptions
- Measure quality

## Workflow
1. Understand task
2. Plan
3. Implement
4. Validate
5. Document

## Checklist
- [ ] Requirements satisfied
- [ ] Tests updated
- [ ] Documentation updated
- [ ] Decisions recorded

## Anti-patterns
- Monolithic prompts
- Hidden assumptions
- Untraceable insights
- Duplicate logic

## Definition of Done
Feature is implemented, tested, documented, and traceable to business goals.
