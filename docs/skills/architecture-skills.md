# Architecture Skills

**Version:** 1.0

**Purpose:** Define the architectural principles, design patterns, system boundaries, and engineering standards for building AI-powered product intelligence platforms.

**Applies To:**

* AI Pipelines
* Voice of Customer Platforms
* Review Discovery Engines
* Product Analytics Systems
* Multi-agent AI Applications

---

# Mission

Architecture exists to enable change.

The goal is not to build the perfect system.

The goal is to build a system that remains understandable, maintainable, extensible, and reliable as requirements evolve.

Optimize for adaptability over initial speed.

---

# Architecture Philosophy

Every architectural decision should improve one or more of:

* Maintainability
* Modularity
* Reliability
* Testability
* Scalability
* Observability
* Replaceability

Avoid architecture that optimizes only for implementation speed.

---

# Guiding Principles

Every component should have:

* One responsibility
* Clear interfaces
* Independent testing
* Explicit dependencies
* Well-defined inputs
* Well-defined outputs

Favor composition over coupling
